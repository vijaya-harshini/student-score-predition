# Student Score Prediction

A web-based application to predict student performance using machine learning.

# Live Demo
https://student-score-predition-1.onrender.com/

# Features

* Predict student score based on input
* Inputs include study hours, sleep hours, and previous marks
* Real-time prediction
* Simple and interactive interface
* Data visualization using charts

# Tech Used

* Python
* Flask
* Scikit-learn
* Pandas
* Matplotlib
